# Survival Analysis in Oncology — Lung Cancer

## Overview

A complete survival analysis of advanced lung cancer data (NCCTG cohort, 228 patients), conducted in R. This project covers the standard biostatistical workflow in oncology, from descriptive exploration to multivariable modeling.

## Methods

- Descriptive statistics (Table 1) with sex-based comparison
- Kaplan-Meier survival curves with log-rank tests (overall, by sex, ECOG score, age group)
- Univariable and multivariable Cox proportional hazards models with Hazard Ratio estimation
- Adjusted forest plot
- Proportional hazards assumption testing (Schoenfeld residuals)
